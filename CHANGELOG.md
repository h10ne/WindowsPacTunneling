# Changelog

### Новое

- Импорт конфигураций **AmneziaWG** (`.conf`) на вкладке «Прокси» — файл или перетаскивание, сохранение в общий список, запуск локального прокси

- Окно не зависает при старте, при запуске/остановке прокси и при включении/отключении PAC
- Кнопка прокси показывает «Запуск...» / «Остановка...» на время операции
- Проверка прокси не застревает в «Проверка» дольше 12 секунд
- Пинг скрыт для VPN-конфигов Amnezia
